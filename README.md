## What is this

This is a test project, was created for work with mongodb's pagination.

## How to run

1. clone the project
2. run the following commands
```bash
npm i

cp ./config/example.json ./config/default.json

docker-compose up -d

npm run start
```

## How to run tests

run the following commands
```bash
cp ./config/example.json ./config/test.json

npm run test
```
